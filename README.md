# MusiCat


O **MusiCat** é um sistema web focado na organização e gestão de repertórios musicais para missas e celebrações litúrgicas. Desenvolvido com foco no conceito *Mobile First*, o objetivo é permitir que músicos tenham acesso rápido as playlists de celebrações, letras, momentos da missa e áudios de forma rápida e acessível.

## 🚀 Funcionalidades (Em Desenvolvimento)

- **Gestão de Playlist:** Adicionar, editar e remover celebrações e músicas das playlists.
- **Organização por Momento:** Categorização automática (Entrada, Salmo, Ofertório, Comunhão, etc.).
- **Visualizador de Letras:** Interface limpa e de alto contraste para leitura durante a celebração.
- **Mini Player Integrado:** Link direto para o YouTube para consulta rápida de melodia.
- **Segurança:** Sistema de login restrito para os membros do ministério.

## 🛠️ Tecnologias Utilizadas

- **Backend:** Python com Flask
- **Frontend:** HTML5, CSS3
- **Banco de Dados:** PostgreSQL
- **Ícones:** Font Awesome 6.0
